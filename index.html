Creare Mundum

  Bienvenue sur Creare Mundum, le projet d'un monde libre!

Objectif:

Creare Mundum est un projet Open Source visant à créer un monde de jeux de rôle libre de droit. Il ne privilégie aucune époque et est adaptable à toutes les règles. Il est rédigé à l'aide de LaTex (un langage de mise en forme de document), distribué sous format pdf et soumis à la licence CC-BY-SA.

Téléchargement:

Vous le voulez, le voila:
  -> La version "lite" (le pdf principal uniquement): Creare_Mundum.pdf
  -> La version "full" (avec les notes de Milan de Garabe, les musiques, les images, les programmes...): crearemundum-master.zip

Participation:

Le projet est hébergé sur GitHub (https://github.com/nobrakal/crearemundum) Plus clairement, pour participer, plusieurs moyens:
  • Dites vos impressions: "Mais le roi gobelin sait qu'il va se faire attaquer, vous le dites deux paragrapghes plus tôt!" Si vous repérez une erreur, faites-nous en part ici! Elle sera corrigé dans les plus brefs delais :)
  • Soumettez-nous des scénarios, des cartes, des description, ce que vous pensez intéressant pour le projet, en créant une issue. Aucunes connaissances en jeu de rôle sur table n'est requise, il nous faut juste des idées!
  • Vous connaissez le LaTex? Aidez à la mise en forme et améliorez le code en créant un pull request.
Si vous avez un quelconque problème, contactez moi:  nobrakal[arobase]gmail.com
